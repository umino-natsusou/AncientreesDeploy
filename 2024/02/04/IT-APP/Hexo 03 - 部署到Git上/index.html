Hexo 03 - 部署到Git上

发表于 2024-02-04 分类于 IT-APP 阅读次数：

1. deployer发布

  1. 将项目上传到git
  2. 项目内安装hexo deployer，npm install hexo-deployer-git --save
  3. 编辑_config.yml，新增deploy，并且更改url地址
  4. 输入 hexo d 发布

2. 公共仓库和私有仓库协同发布

  1. hexo源码上传到私有仓库，令建一个公共仓库用来发布page
  2. 用github工作流来简化发布流程：.github/workflows目录下新建自定义yml后缀的文件，写上如下
